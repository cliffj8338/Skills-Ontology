## Known Syntax Pitfall (learned today)
Avoid string interpolation with escaped quotes in HTML onclick attributes. This pattern breaks in Chrome even when `node --check` passes:
```js
// BROKEN — \' terminates the string in V8 strict mode
pickerHtml += '<button onclick="_fn(\'' + variable + '\')">';

// SAFE — use data attributes
pickerHtml += '<button data-val="' + variable + '" onclick="_fn(this.dataset.val)">';
```
